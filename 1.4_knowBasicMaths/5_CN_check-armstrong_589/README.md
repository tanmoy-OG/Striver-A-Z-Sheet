# Problem Statement

For given a number 'n', return 'true' if number is Armstrong number, return
'false' otherwise.

Armstrong number is a number with 'k' digits, who's sum of each digits raised to
the power of 'k' equals the number itself.

# Example 1:

Input: 'n'=371

Output: 'true'

# Ecample 2:

Input: 'n'=103

Output: 'false'

# Constraints

1 <= ‘n’ <= 10^9

Time Limit: 1 sec
