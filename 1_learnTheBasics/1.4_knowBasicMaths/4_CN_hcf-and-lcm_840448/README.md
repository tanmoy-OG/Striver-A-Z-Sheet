# Problem Statement

For given two numbers 'n' and 'm', calculate the gcd(n, m).

Greatest common divisor is the maximum number that perfectly divides both 'n' and 'm'.

# Example 1:

Input: 'n'=6, 'm'=4

Output: 2

# Ecample 2:

Input: 'n'=9, 'm'=6

Output: 3

# Constraints

0 <= ‘n’ <= 10^5

Time Limit: 1 sec
